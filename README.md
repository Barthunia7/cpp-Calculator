# cpp-Calculator

## Description
A simple calculator application built using C++ performing basic arithmetic operations like addition, subtraction,and division.

## Features
- + Addition
- - Subtraction
- * Multiplication
-  / Division
- Menu-driven interface

## Technologies used
-C++

## How to run
- Clone the repsitory.
- Open the project in any C++ compiler(VS Code/CodeBlocks)
- Compile the program
- Run the program
